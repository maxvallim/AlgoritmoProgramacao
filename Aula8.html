Tipos e Declarações de Variáveis em C++

Contextualização

Em C++, variáveis são usadas para armazenar dados que podem ser manipulados pelo programa. Declarar variáveis corretamente é essencial para garantir que o código seja eficiente e fácil de entender. Aqui estão alguns pontos importantes sobre declarações de variáveis:

  • Onde declarar: Variáveis podem ser declaradas dentro de funções (variáveis locais) ou fora de funções (variáveis globais). Variáveis locais são acessíveis apenas dentro da função onde foram declaradas, enquanto variáveis globais são acessíveis em todo o programa.
  • Por que declarar: Declarar variáveis permite ao compilador alocar memória e definir o tipo de dados que a variável irá armazenar. Isso ajuda a evitar erros e a garantir que o programa funcione corretamente.
  • Variáveis públicas e privadas: Em classes, variáveis podem ser declaradas como públicas (`public`), privadas (`private`) ou protegidas (`protected`). Variáveis públicas são acessíveis de fora da classe, enquanto variáveis privadas são acessíveis apenas dentro da classe. Variáveis protegidas são acessíveis dentro da classe e em classes derivadas.

1. `int`

Tipo de dado inteiro.

int idade = 25;
std::cout << "Idade: " << idade << std::endl;

2. `float`

Tipo de dado ponto flutuante de precisão simples.

float altura = 1.75f;
std::cout << "Altura: " << altura << " metros" << std::endl;

3. `double`

Tipo de dado ponto flutuante de precisão dupla.

double peso = 70.5;
std::cout << "Peso: " << peso << " kg" << std::endl;

4. `char`

Tipo de dado caractere.

char inicial = 'A';
std::cout << "Inicial: " << inicial << std::endl;

5. `bool`

Tipo de dado booleano (verdadeiro ou falso).

bool isAdulto = true;
std::cout << "É adulto: " << std::boolalpha << isAdulto << std::endl;

6. `string`

Tipo de dado para cadeias de caracteres (necessita da biblioteca ``).

#include <string>

std::string nome = "João";
std::cout << "Nome: " << nome << std::endl;

7. `const`

Declara uma variável constante, cujo valor não pode ser alterado.

const int diasNaSemana = 7;
std::cout << "Dias na semana: " << diasNaSemana << std::endl;

8. `unsigned`

Declara uma variável sem sinal (apenas valores positivos).

unsigned int idade = 25;
std::cout << "Idade: " << idade << std::endl;

9. `long`

Declara uma variável inteira longa.

long populacao = 7000000000;
std::cout << "População: " << populacao << std::endl;

10. `short`

Declara uma variável inteira curta.

short ano = 2025;
std::cout << "Ano: " << ano << std::endl;

11. `auto`

Permite ao compilador deduzir automaticamente o tipo da variável.

auto pi = 3.14159;
std::cout << "Valor de pi: " << pi << std::endl;

12. `enum`

Declara um tipo enumerado.

enum Cor { VERMELHO, VERDE, AZUL };
Cor minhaCor = VERDE;
std::cout << "Cor: " << minhaCor << std::endl;

13. `struct`

Declara uma estrutura.

struct Pessoa {
    std::string nome;
    int idade;
};

Pessoa pessoa1;
pessoa1.nome = "Maria";
pessoa1.idade = 30;
std::cout << "Nome: " << pessoa1.nome << ", Idade: " << pessoa1.idade << std::endl;

14. `typedef`

Define um novo nome para um tipo existente.

typedef unsigned int uint;
uint idade = 25;
std::cout << "Idade: " << idade << std::endl;

15. `decltype`

Deduz o tipo de uma expressão.

int x = 10;
decltype(x) y = 20;
std::cout << "y: " << y << std::endl;

16. `extern`

Declara uma variável que é definida em outro arquivo.

extern int contador;
std::cout << "Contador: " << contador << std::endl;

17. `static`

Declara uma variável com duração de armazenamento estática.

static int contador = 0;
contador++;
std::cout << "Contador: " << contador << std::endl;

18. `volatile`

Indica que uma variável pode ser modificada de maneiras não previstas pelo programa.

volatile int sinal;
sinal = 1;
std::cout << "Sinal: " << sinal << std::endl;

19. `mutable`

Permite que um membro de uma classe seja modificado mesmo que a instância da classe seja constante.

class MinhaClasse {
public:
    mutable int contador = 0;
};

const MinhaClasse obj;
obj.contador++;
std::cout << "Contador: " << obj.contador << std::endl;

20. `nullptr`

Representa um ponteiro nulo.

int* ptr = nullptr;
std::cout << "Ponteiro: " << ptr << std::endl;
