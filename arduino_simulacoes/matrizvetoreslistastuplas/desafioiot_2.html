Tutorial: Monitoramento de Jardim com Lista

Diferenças entre Lista, Matriz e Array

  • Array: Estrutura de dados estática. Ideal para manipulação simples e com número fixo de elementos. Sua principal limitação é a falta de flexibilidade.
    Exemplo: int valores[5] = {10, 20, 30, 40, 50};
  • Matriz: Um array de múltiplas dimensões. Útil para armazenar dados em formato tabular, mas não é prático para operações que exigem manipulação dinâmica.
    Exemplo: int matriz[3][3] = {{1, 2, 3}, {4, 5, 6}, {7, 8, 9}};
  • Lista (`std::list`): Estrutura de dados dinâmica da STL. Oferece maior flexibilidade, permitindo adicionar ou remover elementos facilmente. Ideal para cenários em que o número de dados não é fixo.
    Exemplo: list<int> valores = {10, 20, 30, 40, 50};

Explicação das Bibliotecas

  • #include <iostream>: Biblioteca para entrada e saída de dados. Permite exibir mensagens no terminal com cout.
  • #include <list>: Biblioteca da STL que fornece o tipo std::list. Usada para manipulação dinâmica de coleções de dados.
  • #include <string>: Permite manipular textos, caso seja necessário trabalhar com dados mais complexos (não apenas números).

Código do Programa

#include <iostream>
#include <list>
using namespace std;

int main() {
    // Lista simulando os dados de umidade do solo
    list<int> umidades = {35, 42, 27, 50, 15, 60, 39};

    // Processar cada valor de umidade
    int linhaAtual = 1;
    for (int umidade : umidades) {
        cout << "Linha " << linhaAtual << ": ";
        if (umidade < 40) {
            cout << "Umidade " << umidade << "% -> Irrigação acionada." << endl;
        } else {
            cout << "Umidade " << umidade << "% -> Umidade suficiente." << endl;
        }
        linhaAtual++;
    }

    cout << "Monitoramento concluído com sucesso!" << endl;

    return 0;
}
