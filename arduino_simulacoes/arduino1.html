IoT

Simulação de Controle

Vamos desenvolver um algoritmo simples em C++ para simular o controle de um sistema IoT com base na leitura de dados de um arquivo .csv. O sistema detectará se a carga está acima ou abaixo do limite definido e acionará os alertas ou registrará os dados normalmente.

Código:

#include <iostream>
#include <fstream>
#include <string>
#include <sstream>

using namespace std;

// Define o limite de peso
const float LIMITE_PESO = 50.0; // ajuste conforme necessário

void processarDados(const string& arquivoCSV) {
    ifstream arquivo(arquivoCSV);
    string linha;

    if (!arquivo.is_open()) {
        cout << "Erro ao abrir o arquivo CSV." << endl;
        return;
    }

    while (getline(arquivo, linha)) {
        stringstream ss(linha);
        float peso;

        ss >> peso;

        if (ss.fail()) {
            cout << "Erro ao ler o peso da linha: " << linha << endl;
            continue;
        }

        if (peso > LIMITE_PESO) {
            cout << "ALERTA! Peso acima do limite: " << peso << " kg." << endl;
        } else {
            cout << "Registro normal. Peso: " << peso << " kg." << endl;
        }
    }

    arquivo.close();
}

int main() {
    string nomeArquivo = "dados.csv"; // Substitua pelo nome do seu arquivo .csv

    cout << "Processando dados do arquivo: " << nomeArquivo << endl;

    processarDados(nomeArquivo);

    return 0;
}

Explicação:

Arquivo CSV: É esperado que o arquivo .csv contenha linhas com valores numéricos representando pesos. Você pode gerar um arquivo .csv no próprio editor ou programa com os dados fictícios de teste.

Leitura do Arquivo: O algoritmo abre o arquivo .csv, lê os dados linha a linha e tenta convertê-los para o tipo float.

Condicional: Para cada peso lido, verifica se ele está acima ou abaixo do limite definido (LIMITE_PESO). Caso esteja acima, dispara um alerta; caso contrário, registra o peso como normal.

Simulação: Este código não requer hardware como Arduino. Ele é feito apenas para simular o comportamento usando dados do arquivo.
