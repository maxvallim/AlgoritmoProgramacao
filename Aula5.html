Primeiros Passos com C++ no Visual Studio Code

Introdução

Esta apresentação irá guiá-lo através dos primeiros passos para configurar e começar a programar em C++ usando o Visual Studio Code. Incluímos algumas atividades práticas para ajudá-lo a entender melhor a linguagem e o ambiente de desenvolvimento.

Instalando o Visual Studio Code

  1. Acesse o site oficial do Visual Studio Code.
  2. Baixe o instalador para o seu sistema operacional (Windows, macOS ou Linux).
  3. Execute o instalador e siga as instruções na tela.

Instalando o Compilador C++

Para compilar programas em C++, você precisará de um compilador. No Windows, uma opção popular é o MinGW.

  1. Baixe o MinGW do SourceForge.
  2. Execute o instalador e siga as instruções na tela.
  3. Adicione o caminho do MinGW ao PATH do sistema para que o compilador possa ser acessado de qualquer lugar.

Configurando o Ambiente no Visual Studio Code

  1. Abra o Visual Studio Code.
  2. Instale a extensão "C/C++" da Microsoft através da aba de extensões.
  3. Crie uma nova pasta para o seu projeto e abra-a no Visual Studio Code.
  4. Crie um novo arquivo com a extensão `.cpp` (por exemplo, `main.cpp`).

Escrevendo o Primeiro Programa em C++

Vamos começar com um simples programa "Hello, World!" em C++.

#include <iostream>

int main() {
    std::cout << "Hello, World!" << std::endl;
    return 0;
}

Compilando e Executando o Programa

  1. Abra o terminal integrado no Visual Studio Code (View > Terminal).
  2. Navegue até o diretório do seu arquivo `main.cpp`.
  3. Compile o programa usando o comando: 
    g++ -o main main.cpp
  4. Execute o programa compilado usando o comando: 
    ./main

Atividades Práticas

Atividade 1: Calculadora Simples

Crie um programa em C++ que peça ao usuário dois números e exiba a soma, subtração, multiplicação e divisão desses números.

#include <iostream>

int main() {
    double num1, num2;
    std::cout << "Digite o primeiro número: ";
    std::cin >> num1;
    std::cout << "Digite o segundo número: ";
    std::cin >> num2;

    std::cout << "Soma: " << num1 + num2 << std::endl;
    std::cout << "Subtração: " << num1 - num2 << std::endl;
    std::cout << "Multiplicação: " << num1 * num2 << std::endl;
    std::cout << "Divisão: " << num1 / num2 << std::endl;

    return 0;
}

Atividade 2: Estrutura de Controle

Crie um programa que peça ao usuário um número inteiro e determine se ele é par ou ímpar.

#include <iostream>

int main() {
    int num;
    std::cout << "Digite um número inteiro: ";
    std::cin >> num;

    if (num % 2 == 0) {
        std::cout << num << " é par." << std::endl;
    } else {
        std::cout << num << " é ímpar." << std::endl;
    }

    return 0;
}

Atividade 3: Laços de Repetição

Crie um programa que exiba a tabuada de um número fornecido pelo usuário.

#include <iostream>

int main() {
    int num;
    std::cout << "Digite um número para ver a tabuada: ";
    std::cin >> num;

    for (int i = 1; i <= 10; ++i) {
        std::cout << num << " x " << i << " = " << num * i << std::endl;
    }

    return 0;
}
