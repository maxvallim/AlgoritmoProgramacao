Primeiros Passos com C++ no Code::Blocks

Introdução

Esta apresentação irá guiá-lo através dos primeiros passos para configurar e começar a programar em C++ usando o Code::Blocks. Incluímos algumas atividades práticas para ajudá-lo a entender melhor a linguagem e o ambiente de desenvolvimento.

Instalando o Code::Blocks

  1. Acesse o site oficial do Code::Blocks.
  2. Baixe o instalador que inclui o compilador MinGW para o seu sistema operacional (Windows, macOS ou Linux).
  3. Execute o instalador e siga as instruções na tela.

Configurando o Ambiente no Code::Blocks

  1. Abra o Code::Blocks.
  2. Crie um novo projeto selecionando "File > New > Project" e escolha "Console Application".
  3. Siga o assistente de criação de projeto, selecionando C++ como a linguagem de programação.
  4. Escolha um diretório para salvar o projeto e dê um nome ao projeto.
  5. O Code::Blocks criará automaticamente um arquivo principal (`main.cpp`).

Escrevendo o Primeiro Programa em C++

Vamos começar com um simples programa "Hello, World!" em C++.

#include <iostream>

int main() {
    std::cout << "Hello, World!" << std::endl;
    return 0;
}

Compilando e Executando o Programa

  1. Para compilar o programa, clique no ícone "Build and Run" (ou pressione F9).
  2. O Code::Blocks irá compilar o programa e abrir uma janela de console para executar o programa.
  3. Você verá a saída "Hello, World!" na janela de console.

Atividades Práticas

Atividade 1: Calculadora Simples

Crie um programa em C++ que peça ao usuário dois números e exiba a soma, subtração, multiplicação e divisão desses números.

#include <iostream>

int main() {
    double num1, num2;
    std::cout << "Digite o primeiro número: ";
    std::cin >> num1;
    std::cout << "Digite o segundo número: ";
    std::cin >> num2;

    std::cout << "Soma: " << num1 + num2 << std::endl;
    std::cout << "Subtração: " << num1 - num2 << std::endl;
    std::cout << "Multiplicação: " << num1 * num2 << std::endl;
    std::cout << "Divisão: " << num1 / num2 << std::endl;

    return 0;
}

Atividade 2: Estrutura de Controle

Crie um programa que peça ao usuário um número inteiro e determine se ele é par ou ímpar.

#include <iostream>

int main() {
    int num;
    std::cout << "Digite um número inteiro: ";
    std::cin >> num;

    if (num % 2 == 0) {
        std::cout << num << " é par." << std::endl;
    } else {
        std::cout << num << " é ímpar." << std::endl;
    }

    return 0;
}

Atividade 3: Laços de Repetição

Crie um programa que exiba a tabuada de um número fornecido pelo usuário.

#include <iostream>

int main() {
    int num;
    std::cout << "Digite um número para ver a tabuada: ";
    std::cin >> num;

    for (int i = 1; i <= 10; ++i) {
        std::cout << num << " x " << i << " = " << num * i << std::endl;
    }

    return 0;
}
